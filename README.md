# Customer Churn Prediction & Retention Analysis

Predicts customer churn using a Random Forest model (78.6% accuracy, 0.824 ROC-AUC) 
on the IBM Telco dataset, and translates the findings into a segment-specific 
retention playbook.

## Key Findings
- Month-to-month contracts churn at 42.6% vs 2.8% for two-year contracts
- Churn concentrates in the first 18 months of tenure
